## Summary of Application Structure

The application must be written in **English** and submitted through the **NORMA** online portal. The system uses specific character limits (including spaces and line breaks) for each section.

## Critical Content Requirements

### Choice of Database

In your main Project Description, you must include a specific section with the header **"Choice of database"**. Here, you must: \* State the name of the database(s) you have chosen. \* Explain why these specific data sources are necessary for your research. \* Note: You can select a maximum of **two** databases from the shortlisted options.

### Financial Guidelines

-   **Grant Size:** Between **DKK 1 million and DKK 5 million**.
-   **Duration:** Projects must last **1 to 3 years**.
-   **Allowed Costs:** Salaries for technical staff, PhD students, and Postdocs; equipment (up to DKK 200,000/year); and operating expenses.
-   **Exclusions:** You **cannot** use the grant to pay the main applicant's salary or for administrative overhead (rent, electricity, etc.).
